### Sobre Mim

Sou um desenvolvedor focado em construir sistemas **escaláveis e de alta performance**. Minha abordagem combina os princípios de **Clean Architecture** com uma busca constante por eficiência computacional.

* **Sistemas Distribuídos:** Experiência na construção de APIs robustas e microsserviços.
* **Performance First:** Focado em otimização de queries, concorrência e baixo uso de recursos.
* **Tooling:** Usuário assíduo de **Neovim** (e amante de automações via CLI).
* **Data Science:** Conhecimento aplicado em algoritmos de Machine Learning para solução de problemas complexos.
